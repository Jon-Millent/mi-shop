## Install
```
  git clone https://github.com/Jon-Millent/mi-shop.git
  cd mi-shop
  npm install
  npm run dev
```
## Module
* 首页
  * 推荐
  * 手机
  * 智能
  * 电视
  * 电脑
  * 全面屏
  * 生活周边
  * 盒子
  * 艺术
  * 搜索
    * 搜索商品
* 分类
  * 搜索
* 购物车
  * 结算（商品增删）
* 我的
* 商品详情
  * 加入购物车
